Setting up Yomitan for Finnish Language Learning

What is Yomitan?

Yomitan is a browser extension that allows you to hold down a key while reading, hover over a specific word, and then get the wiktionary entry for that word (along with information about its grammatical form).

Using the AnkiConnect Anki add-on, Yomitan can also be configured to automatically create Anki cards (cloze style cards with the rest of the sentence for context) by clicking the green plus icon on the popup.

Note that these cards do have some custom styling.

This means that as you read you can collect flashcards for unfamiliar words that you can then learn using spaced repetition.

Previously, it was quite difficult to use Yomitan with Finnish due to dictionaries needing to reference every single possible form in order for the root word to be correctly detected. Browsers would run into memory issues, meaning that one would need to batch process the dictionary files into smaller chunks before they could be imported. This was suffering and it wasn’t always entirely clear to me whether breaking up the dictionaries meant all words were successfully being detected.

Recently, this memory issue was resolved (Jee!) and one of the larger Finnish dictionaries with excellent detection (but not perfect) can now be imported directly, making it much easier to set up.

Installing the Yomitan Browser Extension

Go to the Yomitan github or wiki site and find the installation link for your relevant browser. I use Firefox myself so the guide is written with this in mind.

Configuring a good Finnish dictionary

The next step on the setup guide for Yomitan is unfortunately no good for learning Finnish! If you install the dictionary that is listed in its recommendations, it seems to only be able to detect nouns in nominative form.

Instead, a Yomitan user created an improved dictionary that can correctly identify the nominative form of a word from many different forms. They posted this dictionary in a link on the Github issue they raised about issues with importing very large dictionaries (the dictionary is only ~160mb compressed, but is 5.4gb uncompressed).

You can download a copy of this dictionary from the linked mega.nz file upload. You can leave the dictionary compressed (unextracted), since Yomitan expects dictionaries to be imported from .zip files.

Once you have downloaded the dictionary, open the Yomitan add-on settings from the shortcut to Yomitan on your browser. You may want to set your language in general settings, and a profile for Finnish if you plan to use Yomitan for multiple languages while you’re here. You might want to change the key that needs to be pressed to show the Yomitan popup too.

Go to the Dictionaries option and then click ‘Configure installed and enabled dictionaries’. Then click Import. Drag and drop the zip file we just downloaded into the import space. Yomitan will then import the dictionary — this will take quite a long time, it will continue to work in the background though.

Once the dictionary is imported, don’t forget to enable it by clicking the toggle.

Linking to Anki for automatic card generation

In order for Yomitan to automatically create Anki cards when requested, you need to download and configure the AnkiConnect add-on. The add-on has the following ID code: 2055492159.

Open the Anki desktop app and open the add-ons menu (Tools -> Add-ons). Click on the “Get add-ons” button and then paste the AnkiConnect add-on code to download the add-on.

On the same add ons menu you will see the local address (127.0.0.1) and the port (e.g. 8765) that the AnkiConnect server is running on. You will need to add this to Yomitan in order for it to interface with Anki.

Go back to Yomitan settings and scroll down to, or click on, the Anki section of the settings menu. On the option “AnkiConnect server address”, you need to paste in the local address and port, e.g. http://127.0.0.1:8765.

While you’re here, I would also recommend adjusting the duplicates setting — personally, I like to detect duplicates but also allow duplicate cards to be made for the same word. I think this is appropriate for Finnish, because you may want multiple cards for different forms of the same word, which wouldn’t be possible because, for example, talossa and talon would flag as duplicates of the word talo.

Anki Card Template

I changed some of my Anki card generation templates, but you can probably leave them how they are if you like. This is what my settings look like on Yomitan’s “Configure Anki Flashcards” option’s menu.

In Anki, my template is set up as the following:

Front Template

<div class="prettify-flashcard">
  <p align='center'>{{Word}}</p>
  <hr> 
  <p align='center'>
    {{cloze-prefix}} <span class="cloze">[...]</span> {{cloze-suffix}}
  </p>
  {{Audio}}
</div>

Back Template

<div class="prettify-flashcard">
  <p align='center'>{{Word}}</p>
  <hr> 
  <p align='center'>
    {{cloze-prefix}} <span class="cloze">{{cloze-body}}</span> {{cloze-suffix}}
  </p>
  {{conjugation}}
  {{Meaning}}<br>
</div
